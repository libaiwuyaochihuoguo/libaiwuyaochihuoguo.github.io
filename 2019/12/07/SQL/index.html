MySQL数据库常用的SQL语句

(注: 在所有操作之前 USE 库名;)

  • DQL–查询语言query
    1. 查询表中单个字段：
      • SELECT 查询列表列名 FROM 表名;
    2. 查询表中多个字段：
      • SELECT 列名1,列名2 FROM 表名;
    3. 查询所有字段：
      • SELECT * FROM 表名;
    4. 查询常量值：
      • SELECT 常量值;e g.SELECT 100
    5. 查询表达式：
      • SELECT 100*98;
    6. 查询方法：
      • SELECT 方法名();
    7. 起别名(把列名换成别名，防止重复)
      • SELECT 100*98 AS 结果;
      • SELECT 列名1 AS 别名1,列名2 AS 别名2 FROM 表名;
    8. 去重：
      • SELECT DISTINCT 列名1 FROM 表名;
    9. 条件查询、模糊查询、排序查询等
  • DML –数据操作语言manipulation
    1. 插入：//列个数需和值个数一致
      • INSERT INTO 表名(列名1，列名2) VALUES(值1，值2);//此方式可插入多行
      • INSERT INTO 表名 SET 列名1=值1，列名2=值2;
    2. 修改：
      • UPDATE 表名 SET 列名1=新值1，列名2=新值2 WHERE 筛选条件;//不筛选即为全局修改
    3. 删除：
      • DELETE FROM 表名 WHERE 筛选条件;//不加筛选条件即为删除整表，加条件可删除具体某行
  • DDL –数据定义语言definition
    1. 库的管理
      • 创建：
        CREATE DATABASE 库名;
      • 修改：//一般不修改，防止出现数据丢失
        ALTER语句
      • 删除：
        DROP DATABASE 库名;
    2. 表的管理
      • 创建：
        CREATE TABLE 表名(列名1 列名1的数据类型, 列名2 列名2的数据类型);
      • 修改：
        1. 修改列名：
          ALTER TABLE 表名 CHANGE COLUMN 旧列名 新列名 新列的数据类型;
        2. 修改列的数据类型：
          ALTER TABLE 表名 MODIFY COLUMN 列名 该列新的数据类型;
        3. 添加列：
          ALTER TABLE 表名 ADD COLUMN 新列名 该列的数据类型;
        4. 删除列：
          ALTER TABLE 表名 DROP COLUMN 列名;
        5. 修改表名：
          ALTER TABLE 表名 RENAME TO 新表名;
      • 删除：
        DROP TABLE 表名;
